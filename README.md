# Chaithanya K Portfolio

A clean personal portfolio website for a Java Full Stack Developer specializing in Spring Boot, React, Microservices, AWS, Kafka, Docker, Kubernetes, and AI/RAG applications.

## Sections

- Home
- About
- Skills
- Experience
- Projects
- Education & Certifications
- Contact

## How to Run

Open `index.html` directly in your browser.

## How to Customize

1. Replace the resume file:
   - Add your resume as `Chaithanya_K_Resume.pdf`

2. Update contact details:
   - Edit email, phone, LinkedIn, and location in `index.html`

3. Add GitHub links:
   - In the Projects section, add buttons for GitHub and live demo links.

## Deploy

You can deploy this website using:
- GitHub Pages
- Netlify
- Vercel
